# 🍔 Burgueria DevClub

Projeto de uma página de hamburgueria desenvolvido com **HTML, CSS e JavaScript**, com foco em **manipulação de arrays e DOM** utilizando métodos modernos do JavaScript.

---

## 🚀 Funcionalidades

O sistema permite interagir com os produtos através de botões:

### 🍔 Mostrar todos os produtos (forEach)

Exibe todos os itens do cardápio na tela.

👉 Utiliza:

* `forEach` para percorrer os produtos
* Manipulação do DOM (`innerHTML`)

---

### 🌱 Filtrar produtos veganos (filter)

Mostra apenas os produtos veganos.

👉 Utiliza:

* `filter` para retornar apenas itens com `vegan = true`

---

### 💸 Aplicar desconto (map)

Aplica **10% de desconto** em todos os produtos.

👉 Utiliza:

* `map` para criar um novo array com os preços atualizados
* `price * 0.9`

---

### 💰 Valor total dos produtos (reduce)

Calcula o valor total de todos os produtos.

👉 Utiliza:

* `reduce` para somar todos os preços
* `toFixed(2)` para formatação

---

## 🧠 Conceitos aplicados

* Manipulação de arrays
* Métodos de array:

  * `filter`
  * `map`
